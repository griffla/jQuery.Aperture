jQuery.Aperture
===============
Rotate all kinds of elements clock- or counterclockwise.

by Joscha Schmidt - http://aperture.joschaschmidt.de/demo

Options
=======
| Option | Default
|:-----------|:------------|
| columns    | 4 |
| easing     | easeOutQuint |
| imgWidth   | 200px |
| imgHeight  | 140px |
| duration   | 1000 |
